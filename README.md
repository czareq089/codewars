# Moje Rozwiązania z CodeWars

Witaj w moim repozytorium zawierającym rozwiązania zadań ze strony CodeWars. Zadania są posegregowane według daty ich wykonania, w odpowiednich folderach.

## Struktura Repozytorium

├── Data -> Nazwa zadania -> program.cs

W każdym folderze znajdują się foldery z nazwami zadań oraz pliki `program.cs` z rozwiązaniami zadań wykonanymi w danym dniu.

## O CodeWars

[CodeWars](https://www.codewars.com/) to platforma, na której programiści mogą doskonalić swoje umiejętności poprzez rozwiązywanie różnorodnych zadań programistycznych. Każde zadanie posiada określony poziom trudności oraz testy, które muszą być przechodzone przez poprawnie napisane rozwiązanie.
